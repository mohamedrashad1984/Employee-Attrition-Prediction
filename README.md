# Employee Attrition Prediction

**Goal:** Predict which employees are at risk of leaving the company using machine learning.

**Tools:** Python, Pandas, Matplotlib, Scikit-learn, SHAP

**Highlights:**
- Exploratory Data Analysis with business insights
- Logistic Regression ML pipeline
- Feature importance & SHAP explainability
- Predicted top 10 high-risk employees for retention strategies

**How to Run:**
1. Download the repository
2. Open `Employee_Attrition_Prediction.ipynb` in Jupyter or Google Colab
3. Install required libraries: `pip install -r requirements.txt` (optional)

**Dataset:** Included small sample CSV
